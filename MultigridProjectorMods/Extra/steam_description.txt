[h1]Multigrid Projector Extra[/h1]

Additional useful functionality and extra convenience features for the projector block.

Despite its name this mod also work with the vanilla game, but some features require the Multigrid Projector plugin.

[b]It is a regular mod which needs to be added to the world[/b]. This is [b]not[/b] a plugin! This is [b]not[/b] a mod version of the Multigrid Projector.

[h2]Features work without the MGP[/h2]
[list]
[*] Projector control panel button to load a self-repair projection for the projector's own mechanical group (ship)
[*] Intuitive manual alignment of the projection with familiar character and block movement
[/list]

[h2]Features require the MGP[/h2]
[list]
[*] None so far
[/list]

More features useful for projectors and console blocks are expected to be added to this mod in the future.

[i]Upon loading the self-repair projection "Keep projection" is automatically enabled.[/i]
